Mathmatical operation
==

#### 정수를 아무리 정수랑 나누어도 항상 float이 나온다
> print(3 / 2)   
> print(3 / 1)
```
1.5
3.0
```

#### 나눗셈을 뺴고 몫만 가르쳐준다.
> print(3 // 2)
```
1
```
>print(7 // 2)
```
3
```
#### 지수
>print(3 ** 3) > 3의 3의 3승
```
27
```
#### 나눗셈 > 소숫점도 가능하다
>print(60 % 13)
>print(12.5 % 5.5)
```
8
1.5
```
