복합 조건절
==

#### 복합 조건절
+ 모든 조건이 개별적으로 검사된다.
+ 참인 조건이 모두 실행이 된다.

````
x = 10

if x > 5:
    print("5보다 큼")
if x < 20:
    print("20보다 작음")
if x == 10:
    print("10과 같음")
````
>5보다 큼   
>20보다 작음   
>10과 같음

```
#### 거짓인 조건은 건너 뛰고 출력한다.

    print("5보다 큼")
if x > 20:
    print("20보다 큼")

```
>5보다 큼
