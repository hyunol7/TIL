JSON(JavaScript Object Notation)
==

#### JSON
+ JSON은 Key, Value로 이루어진 데이터를 전달하기 위한 표준 포
+ JSON 데이터를 쉽게 사용(데이터 추출)할 수 있도록 해주는 함수
+ 데이터를 저장할 때, JSON 형태를 종종 활용

```
{
  "name": "Kyle",
  "age": 35,
  "address": {
    "city": "Seoul",
    "country": "South Korea"
  },
  "hobbies": ["reading", "walking", "coding"]
 }
```
