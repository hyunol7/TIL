지표 문제 접근 순서
==

### 목적
+ 현재 시점에서 사용자의 행동이나 특징을 요약하고 분류하는 것
+ 마케팅, 사용자 세분화, 서비스 개선을 위한 "상태 스냅샷"

### 패턴
+ 기준 정하기 > 날짜, 유저 ID, 이벤트명 등
+ GROUP BY + COUNT / SUM / AVG 등 사용 > 유저별 집계, 특정 조건으로 CASE WHEN 분류
+ 시각화/정렬 가능하도록 가공



리텐션 문제 접근 순서
===

### 목적
+ 특정 날짜나 주차에 앱을 처음 쓴 유저가 그 이후에도 다시 돌아오는지를 추적하는 것
+ 지속성, 중독도, 서비스 매력도 판단
